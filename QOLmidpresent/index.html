QoL 활동사항

서버관리, 컨텐츠관리, 업무프로세스 확립, 내부 교육, 기능 구현

서버관리

컴교과 홈페이지는 XE코어로 이루어져 있습니다.
XE는 아파치/PHP 기반이며, 게시판에 최적화되어 있습니다.
저희 QoL 집행부에서는 현재 AWS EC2 마이크로 인스턴스를
사용중이며, 이를 통해 편리한 웹환경을 제공하고 있습니다.
해당 서버 인스턴스 및 엔진 관리는 항시 진행중입니다.

컨텐츠관리

홈페이지에 등록된 학과소개를 비롯한 집행부/소모임/활동 소개 등의 컨텐츠를 항상 최신정보로 유지될 수 있도록 QoL 컨텐츠팀에서 관리하고 있습니다.

업무프로세스 확립

QoL은 2017년 1학기 현재 학기부터 처음 활동을 시작한 신설부서입니다. 그러므로 무엇보다 앞으로 지속적인 인수인계 및 발전을 위해선 정립된 업무프로세스가 필수입니다. 이를 위해 어느 신입멤버든, 부서 리더가 누구이든 상관없이 효율적으로 개발할 수 있는 환경을 조성하였습니다.

내부교육

QoL은 자유로운 인재풀을 지향합니다. 다양한 관심사의 학과 동기/선후배들이 모인 QoL은 구성원들의 전문성 함양을 위해 프로그래밍/협업도구 등에 관한 내부교육을 실시합니다.

기능구현

QoL의 설립목적이자 심장이라고도 할 수 있는 기능 구현은 매우 중요합니다. 학과 구성원들, 사용자들에게 편의를 제공할 수 있는 홈페이지 기능을 개발합니다.

서버자료 git 통합

기존의 물리적 터미널 혹은 SSH등으로 불편하게 관리해야 했던 홈페이지가 Git을 이용하여 아무 장소에서나 누구나 권한이 있으면 쉽게 관리할 수 있게 되었으며, 버전관리가 되므로 문제가 생길시 롤백도 용이해졌습니다. 무엇보다 서버가 유실된다고 해서 홈페이지를 아예 잃어버리거나 졸업생 선배에게 혹시 홈페이지 파일이 있는지 물어볼 필요 없이, QoL Github 리포지토리(오픈소스)를 접근하면 됩니다.
