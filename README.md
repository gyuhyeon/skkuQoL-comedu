### 추가기능 구현

추가기능은 주로 PHP로 XE코어와 호환되는 방식으로 구현하였으며, 해당 프로젝트들은 /qol 폴더에 있습니다.

추후 프로젝트는 nodejs를 사용할 가능성도 열어두고 있습니다.
